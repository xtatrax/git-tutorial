## そもそもgitとは

バージョン管理システム = 誰が、何を、いつ、どう、変更したかを管理するツールです。

![git-describe](./github/images/github-log-describe.png)

### git は それ単体でサーバー機能を提供できます。

GitHub などのサービスを使わなくても、完全ローカルや、社内や家庭内などのローカルネットワークでも使う事ができます。
